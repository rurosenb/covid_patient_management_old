## Technologies
Project is created with:
* Server
  * Node(.js)
  * Express(.js)
  * JSON
* Client
  * React
  * react-router-dom
  * axios
* Databases
  * MongoDb/Mongoose
	
## Setup
To run this project, install it locally using npm:

```
$ cd ../lorem
$ npm install
$ npm start
```
